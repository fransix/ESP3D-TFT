# BSP: ESP32-Azure-IoT-Kit
Board support package consists of:
* uSD card interface
* I2C interface
* LEDs, button and buzzer interface
* Sensors software components: MAG3110, HTS221, BH1750, FBM320 and MPU6050
* Display software component: SSD1306
